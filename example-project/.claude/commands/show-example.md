/kiroコマンドの出力例を表示します。

## requirements.mdの例
```markdown
# TODOアプリケーション要件定義書

## ユーザーストーリー

### ストーリー1: タスクの作成
**As a** ユーザー
**I want** 新しいタスクを作成できる
**So that** TODOリストを管理できる

**受け入れ基準:**
- [ ] タスクタイトルを入力できる
- [ ] Enterキーでタスクが追加される
- [ ] 空のタイトルでは作成できない
```
